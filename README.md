# Community Detection with Graph Neural Networks via QUBO Formulation

Bachelor's thesis code repository. Combines QUBO formulations of modularity
maximisation (Wang et al. 2024) with an unsupervised graph neural network
solver (QIGNN) for community detection. The thesis is in [`thesis/`](thesis/).

## Quick start

### Environment

Python 3.12 with PyTorch 2.2, DGL 2.2, NetworkX, scikit-learn:

```bash
pip install -r requirements.txt
```

### Reproduce a single experiment

The runners are parallel and stream results to CSV. A "quick" mode runs a
minimal smoke test (~5 seconds):

```bash
# Smoke test — 3 runs total
python experiments/run_hp_sweep.py --quick --workers 2

# Full hyperparameter sweep (3 graphs × 24 configs × 5 shots = 360 runs, ~15 min)
python experiments/run_hp_sweep.py --workers 8

# Hierarchical algorithm on all 13 graphs (~3 hours)
python experiments/run_hierarchical.py --workers 8
```

Outputs go to `results/<experiment>.csv` and `results/<experiment>_agg.csv`.

### Sanity-check the QUBO formulation

```bash
python src/qubo_modularity.py
```

Runs three unit tests on the QUBO formulas 5 and 6 from Wang et al. 2024:
structural checks, equivalence between formulations on Karate, and exhaustive
argmin agreement on a small 5-node graph.

## Method in one paragraph

The thesis adapts the QIGNN unsupervised QUBO solver (Schuetz et al. 2022,
extended in [qignn2026]) to community detection. Two QUBO formulations from
Wang et al. 2024 are used:

- **Formula 6** (single-bit, $k=2$): $Q = -B$, where $B$ is the modularity
  matrix. Converges deterministically; no constraint penalty needed.
- **Formula 5** (one-hot, $k > 2$): $nk$ binary variables with a soft
  one-hot penalty. Empirically prone to trivial collapse.

The main contribution is a **hierarchical algorithm** that recursively
applies formula 6 with global modularity gain as the stopping criterion.
It is empirically deterministic (σ < $10^{-4}$ across seeds on 10 of 13
benchmarks) and matches or exceeds Louvain in NMI on 10 of 13 benchmarks.

## Key results

See `thesis/main.tex` chapter 4 for the full evaluation. The two headline
findings are:

- **Bipartition formulation (formula 6) is deterministic**: σ across 20 random
  seeds is below $10^{-4}$ on 5 of 7 binary benchmarks (and ≤ 0.007 on the
  remaining two). NMI exceeds Louvain on 5 of 7 benchmarks, with the largest
  improvement being +0.333 on Dolphins.
- **Hierarchical algorithm**: matches Louvain in modularity within 0.019 on
  all 13 benchmarks; matches or exceeds Louvain in NMI on 10 of 13. Inherits
  the determinism of formula 6 (σ ≤ 0.005 on every benchmark, < $10^{-4}$ on
  most).
